# Тестирование функциональности объявлений с помощью Postman
## Регистрация и авторизация
### Регистрация пользователя
### **Method: POST**
### **URL: /api/auth/register**
### **Body (raw JSON):**
{\
`    `"email": "test@example.com",\
`    `"password": "password123",\
`    `"role": "USER"\
}

- **Ожидаемый ответ**: 200 OK с JWT токеном
### Авторизация (получение токена)
### **Method: POST**
### **URL: /api/auth/login**
### **Body (raw JSON):**
{\
`    `"email": "test@example.com",\
`    `"password": "password123"\
}
## Тестирование функциональности объявлений
### Создание объявления
### **Method: POST**
### **URL: /api/listings/add**
### **Headers:**
Authorization: Bearer authToken

**Body** (raw JSON):

{\
`    `"title": "iPhone 14 Pro",\
`    `"brand": "Apple",\
`    `"price": 89990.00,\
`    `"condition": "New"\
}

**Формат создания объявления:**

- title (string): Название объявления (обязательно)
- brand (string): Бренд техники (обязательно)
- price (decimal): Цена в формате числа с плавающей точкой (обязательно)
- condition (enum): Состояние техники, возможные значения:
  - "New" - Новое
  - "Used" - Б/у

**Ожидаемый ответ**:

{\
`    `"id": 1,\
`    `"title": "iPhone 14 Pro",\
`    `"brand": "Apple",\
`    `"price": 89990.00,\
`    `"condition": "New",\
`    `"status": "Active",\
`    `"ownerId": 1,\
`    `"createdAt": "2024-03-09T15:30:00",\
`    `"updatedAt": "2024-03-09T15:30:00"\
}
### Получение всех объявлений
### **Method: GET**
### **URL: /api/listings?page=0&size=10&sortBy=createdAt&direction=desc**
**Параметры**:

- `        `page - номер страницы (начинается с 0)
- `        `size - количество элементов на странице
- `        `sortBy - поле для сортировки (createdAt, price, title)
- `        `direction - направление сортировки (asc, desc)
### Получение конкретного объявления
**Method**: GET

**URL**: /api/listings/{listingId}\
**Headers**: Без авторизации
### Фильтрация по бренду
**Method**: GET\
**URL**: /api/listings/brand/Apple\
**Headers**: Без авторизации
### Фильтрация по состоянию
**Method**: GET\
**URL**: /api/listings/condition/New\
**Headers**: Без авторизации

**Возможные значения condition**:

- `        `New - Новые
- `        `Used - Б/у
### Фильтрация по диапазону цен
### **Method: GET**
**URL**: /api/listings/price-range?minPrice=50000&maxPrice=100000\
**Headers**: Без авторизации\
**Параметры**:

- `        `minPrice - минимальная цена
- `        `maxPrice - максимальная цена
### Расширенная фильтрация
### **Method: POST**
**URL**: /api/listings/filter\
**Body** (raw JSON):

{\
`    `"brand": "Apple",\
`    `"condition": "New",\
`    `"minPrice": 50000,\
`    `"maxPrice": 100000\
}
### Получение объявлений пользователя**        
### **Method: GET**
**URL**: /api/listings/user\
**Headers**:

Authorization: Bearer authToken
### Обновление объявления
**Method**: PUT\
**URL**: /api/listings/{{listingId}}\
**Headers**:

Authorization: Bearer authToken

**Body** (raw JSON):

{\
`    `"title": "iPhone 14 Pro Max",\
`    `"brand": "Apple",\
`    `"price": 99990.00,\
`    `"condition": "New"\
}
### Удаление объявления
**Method**: DELETE\
**URL**: /api/listings/{{listingId}}\
**Headers**:

Authorization: Bearer authToken
## Тестирование различных сценариев
**Пример 1: Новое устройство**

{\
`    `"title": "MacBook Pro 16\"",\
`    `"brand": "Apple",\
`    `"price": 150000.00,\
`    `"condition": "New"\
}

**Пример 2: Б/у устройство**

{\
`    `"title": "iPhone 12",\
`    `"brand": "Apple",\
`    `"price": 45000.00,\
`    `"condition": "Used"\
}

**Пример 3: Другой бренд**

{\
`    `"title": "Galaxy S23",\
`    `"brand": "Samsung",\
`    `"price": 65000.00,\
`    `"condition": "New"\
}

**Фильтрация по бренду Samsung**:

GET {{baseUrl}}/api/listings/brand/Samsung

**Фильтрация по состоянию Used**:

GET {{baseUrl}}/api/listings/condition/Used

**Фильтрация по цене от 50000 до 80000**:

GET {{baseUrl}}/api/listings/price-range?minPrice=50000&maxPrice=80000

**Комбинированная фильтрация**:

POST {{baseUrl}}/api/listings/filter\
{\
`    `"brand": "Apple",\
`    `"condition": "Used",\
`    `"minPrice": 30000,\
`    `"maxPrice": 70000\
}
### Тестирование пагинации
**Первая страница (10 элементов)**:

GET {{baseUrl}}/api/listings?page=0&size=10

**Вторая страница (5 элементов)**:

GET {{baseUrl}}/api/listings?page=1&size=5
### Тестирование сортировки
**Сортировка по цене (возрастание)**:

GET {{baseUrl}}/api/listings?sortBy=price&direction=asc

**Сортировка по дате создания (убывание)**:

GET {{baseUrl}}/api/listings?sortBy=createdAt&direction=desc
